#### _February 15th, 2023_

activate the env

```
source env/bin/activate
```

Several commands:

```
django-admin startproject [project-name]

python manage.py startapp [app-name]

python manage.py runserver

python manage.py migrate
```

Creat admin account:

```
python manage.py createsuperuser
```
